# ERP Responsible AI

Enterprise-style ERP learning and voice coaching platform built with Streamlit.

## One-click Render deployment

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/sawantmangesh89-arch/erp-responsible-ai)

This repository already includes `render.yaml`, so Render can provision the service directly from this repo.

- `voice_erp_coach.py`
